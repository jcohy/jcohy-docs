== 介绍

Spring Boot Maven 插件在  https://maven.org[Apache Maven] 中提供了对 Spring Boot 支持. 它允许您打包可执行的 jar 或 war 文件,运行 Spring Boot 应用程序,生成构建信息以及在运行集成测试之前启动 Spring Boot 应用程序.
