[[prometheus]]
= Prometheus (`prometheus`)

`prometheus` 端点提供了由 Prometheus 服务器所需的格式 Spring 启动应用程序的指标.

当使用适当的 `Accept` 头调用时，端点还可以生成  `application/openmetrics-text;version=1.0.0` ，如以下基于 curl 的示例所示：
