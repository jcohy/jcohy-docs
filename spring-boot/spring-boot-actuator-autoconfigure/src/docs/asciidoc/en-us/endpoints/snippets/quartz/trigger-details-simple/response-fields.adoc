|===
|Path|Type|Description

|`+simple+`
|`+Object+`
|Simple trigger specific details.

|`+simple.interval+`
|`+Number+`
|Interval, in milliseconds, between two executions.

|`+simple.repeatCount+`
|`+Number+`
|Number of times the trigger should repeat, or -1 to repeat indefinitely.

|`+simple.timesTriggered+`
|`+Number+`
