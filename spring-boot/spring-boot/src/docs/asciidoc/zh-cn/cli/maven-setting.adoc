[[cli.maven-setting]]
== 使用 settings.xml 配置 CLI
Spring Boot CLI 使用 Maven 的依赖解析引擎 Aether 来解决依赖.  CLI使用 `~/.m2/settings.xml` 中的 Maven 配置来配置 Aether.  CLI 遵循以下配置设置:

* Offline
* Mirrors
* Servers
* Proxies
* Profiles
** Activation
** Repositories
* Active profiles

有关更多信息,请参见 https://maven.apache.org/settings.html[Maven 的设置文档].
