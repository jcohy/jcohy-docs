= GraalVM Native Image 支持
include::attributes.adoc[]

https://www.graalvm.org/native-image/[GraalVM Native Images] 是独立的可执行文件，可以通过提前处理已编译的 Java 应用程序来生成。
原生镜像通常具有较小的内存占用，并且启动速度比 JVM 对应项更快。
