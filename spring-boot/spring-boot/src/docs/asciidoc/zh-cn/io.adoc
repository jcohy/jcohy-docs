= IO
include::attributes.adoc[]

大多数应用程序在某些时候需要处理输入和输出问题。 Spring Boot 提供实用程序和与一系列技术的集成，以在您需要 IO 功能时提供帮助。 本节涵盖标准 IO 功能（例如缓存和验证）以及更高级的主题（例如调度和分布式事务）。 我们还将介绍调用远程 REST 或 SOAP 服务以及发送电子邮件。
