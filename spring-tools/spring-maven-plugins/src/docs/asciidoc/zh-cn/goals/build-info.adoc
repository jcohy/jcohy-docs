[[goals-build-info-parameters-details-time]]
==== `time`
`build.time` 属性的值，格式为 `Instant#parse(CharSequence)`. 默认为 `session.request.startTime`. 要完全禁用 `build.time` 属性, 请使用 `'off'`.
