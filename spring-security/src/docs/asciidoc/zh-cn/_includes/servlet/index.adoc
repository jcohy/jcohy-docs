= Servlet 应用程序

Spring Security使用标准的 Servlet 过滤器与 Servlet 容器集成.  这意味着它可以和使用 Servlet 容器运行的任何应用程序一起使用.  更具体地说,你可以在不使用 Spring,而是基于 Servlet 的应用程序中使用 Spring Security.
