[[publishing_maven:signing]]
= Signing artifacts（签名构件）

<<signing_plugin.adoc#signing_plugin, Signing Plugin>> 可用于对发布的所有构件和元数据文件进行签名，包括 Maven POM 文件和 Ivy 模块描述符。为了使用它：

1. 应用 Signing Plugin
2. 配置 <<signing_plugin.adoc#sec:signatory_credentials,signatory credentials>> — 如下所述
3. 指定要签名的 publications

这是一个配置插件以对 `mavenJava` 进行签名的示例：
