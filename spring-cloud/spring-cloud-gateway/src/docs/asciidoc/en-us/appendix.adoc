== Common application properties

include::_attributes.adoc[]

可以在 `application.properties` 文件或 `application.yml` 文件中指定各种属性，也可以将其作为命令行来指定。

本附录提供了常用 {project-full-name} 属性列表以及对使用这些属性的基础类的引用。

NOTE: 属性可以来自类路径上的其他 jar 文件，因此您不应将此视为详尽列表。此外，您还可以定义自己的属性。
