= Populators  命名空间参考

[[namespace-dao-config]]
== <populator /> element

`<populator />` 元素允许通过 Spring 数据存储库基础结构填充数据存储. footnote:[参阅 <<repositories.adoc#repositories.create-instances.xml>>]

.属性
[options="header", cols="1,3"]
|===============
|名称|描述
|`locations`|从哪里可以找到要从存储库读取对象的文件,应在其中填充.
|===============
