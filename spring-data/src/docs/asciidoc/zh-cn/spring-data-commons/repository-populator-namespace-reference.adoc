= Populators  命名空间参考

[[namespace-dao-config]]
== <populator /> element
`<populator />` 元素允许通过Spring数据存储库基础结构填充数据存储. footnote:[参阅 <<repositories.create-instances.spring>>]

.属性
[options="header", cols="1,3"]
|===============
|名称|描述
|`locations`|从哪里可以找到要从存储库读取对象的文件，应在其中填充.
|===============
